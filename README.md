---SmartDemand — Demand Forecasting for Retail

Predict what sells. Stock what matters. Never run out.

Inspired by Williams Sonoma's retail problem, a supply chain intelligence tool that forecasts product demand across multiple stores and prevents stockouts before they happen.

----What It Does

Predicts units sold for the next 7 days using XGBoost ML
Recommends optimal inventory levels per store & product
Alerts teams on stockout and overstock risks via email
Dashboards for admins and data reviewers with full edit control

---Tech Stack
Python 3.10 · Streamlit · XGBoost · pandas · Plotly · bcrypt
